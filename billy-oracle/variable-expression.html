Data Types

You have already seen a type of data known as strings, which can be sequences of any kind of characters (letters, numbers, symbols, spaces, etc.), such like "The book of: 1 Nephi" and "yellow puppy".

A sequence of number, such as "48212", can be converted to a numeric data type and stored internally as a number; otherwise, the computer doesn't distinguish numbers from a sequence of letters. Once you have converted the String to a number, you can use it in mathematical formulas of expressions.

Mixing Strings And Numbers

A potential problem arises if you try to mix strings and numeric data types together. For example, trying to add a string to an integer produces the following error:

The computer doesn't know how to add strings to numbers, which causes an error. In this case, what you want is for the computer to treat the number stored in the "apple_count" variable as a string instead of a number. It might seem obvious to you that this is the desired behavior, but the computer doesn't try to guess what you want, you must explicitly tell it when you want it to convert one type of data to another.

Converting data from one type to another is called type-casting, or "casting". That's just a fancy way of saying "treat it like a number please". To convert data to an integer, you put "int" before it, and then wrap it in parentheses, like this: "int(data_type)". The same is true for converting it to a floating number with the term "float()" or string with "str()" and so on.
